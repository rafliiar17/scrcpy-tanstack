# ADB Binaries for Bundling

Place the official ADB binaries from Google Platform Tools here for each platform. 
ScrcpyGUI Pro will automatically detect and use these binaries on the corresponding OS.

### Structure:
- `windows/adb.exe`
- `linux/adb`
- `macos/adb`

### Downloads:
- Windows: https://dl.google.com/android/repository/platform-tools-latest-windows.zip
- Linux: https://dl.google.com/android/repository/platform-tools-latest-linux.zip
- macOS: https://dl.google.com/android/repository/platform-tools-latest-darwin.zip

> [!IMPORTANT]
> Ensure the files are named exactly `adb.exe` or `adb` and are in their respective platform folders.
